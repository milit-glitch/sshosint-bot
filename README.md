# Shodan Telegram Bot

Telegram bot for interacting with Shodan API.

## Features
- Search by IP
- Search by query
- Host information

## Installation
```bash
pip install -r requirements.txt
```

## Setup
Create `.env` file:
```
BOT_TOKEN=your_token
SHODAN_API_KEY=your_key
```

## Run
```bash
python main.py
```
